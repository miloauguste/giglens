## Security Scanning

GigLens uses a multi-layer security scanning pipeline on every commit:

| Check | Tool | Frequency |
|-------|------|-----------|
| Static analysis | Android Lint | Every commit |
| Secret detection (source) | Gitleaks | Every commit |
| Secret detection (res files) | Gitleaks | Every commit |
| Hardcoded values | Custom grep | Every commit |
| APK permission audit | aapt | Every commit |
| Dependency CVEs | OWASP Dependency Check | On-demand (OWASP_SCAN=1) |
| APK static analysis | MobSF v4.5.0 | On-demand (MOBSF_SCAN=1) |

## MobSF Score Baseline

### Known False Positives (do not fix)

| Finding | Reason | Status |
|---------|--------|--------|
| `32da061bd771fea748f16e6ab05cfee3` hardcoded secret | Room DB schema identity hash — auto-generated by Room compiler, not a secret | ✅ False positive |
| `73b814adea7d78fa14fec0a9f2ec3b2a` hardcoded secret | Room DB schema identity hash — auto-generated by Room compiler, not a secret | ✅ False positive |
| Hardcoded strings in AppConfig.java | Room/kapt generates Java from Kotlin — DB key name constants flagged incorrectly | ✅ False positive |
| `androidx.profileinstaller.ProfileInstallReceiver` permission | Third-party AndroidX library — not our code | ✅ False positive |

### Debug Build Limitations (auto-fixed on release)

| Finding | Fix |
|---------|-----|
| Application signed with debug certificate | Sign with release keystore |
| `android:debuggable=true` | Release build disables this automatically |

### Accepted Risks (required for app functionality)

| Permission | Why Required |
|-----------|-------------|
| `SYSTEM_ALERT_WINDOW` | Floating pill widget over DoorDash |
| `ACCESS_FINE_LOCATION` | GPS-based geocoding for distance estimation |
| `ACCESS_COARSE_LOCATION` | Fallback location for geocoding |
| `POST_NOTIFICATIONS` | Foreground service notification required by Android |

### Fixed Security Issues

| Date | Issue | Fix |
|------|-------|-----|
| 2026-05-23 | `android:allowBackup=true` | Set to false — prevents ADB data extraction |
| 2026-05-23 | `BootReceiver android:exported=true` | Set to false — BOOT_COMPLETED works without public export |
| 2026-05-23 | `ShareReceiverActivity` unprotected export | Added `MANAGE_DOCUMENTS` permission gate |
